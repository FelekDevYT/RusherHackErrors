# List of RusherHack errors when starting Minecraft

 > [!WARNING]
 > Not all error fixes may be listed here, and if you are unable to fix the error using these instructions, please contact support at support@rusherhack.org or open a ticket on the Discord server.
 
## Terms used in the table:
- **HWID (Hardware ID)** - your computer's unique ID.

| Error code | Description                                                                                         | How to fix                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| ---------- | --------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 0x1000     | An attempt to verify the computer's HWID failed. This happens when changing a PC or its components. | Send an email to support@rusherhack.org with a request to reset your HWID or open a ticket on the Discord server.                                                                                                                                                                                                                                                                                                                                                                                                                               |
| 0x1200     | RusherHack loader can't find authentication files on your drive.                                    | On Linux:<br>Option 1:<br>- Install Flatseal from your store.<br>- Find the Minecraft launcher, and enable All user files.<br>- You should now be able to launch rusherhack, however you may need to request a HWID reset.<br>Option 2:<br>Run the command: sudo flatpak override \<Flatpak Id\> --filesystem=home<br>On Windows:<br>1. Your antivirus may have deleted the rusherhack files (extremely unlikely).<br>2. Reinstall RusherHack using the RusherHack Installer, which you can download from your personal account on the website. |
| 0x1250     | General profile authentication error                                                                | Reinstall RusherHack using the RusherHack Installer, which you can download from your personal account on the website.                                                                                                                                                                                                                                                                                                                                                                                                                          |
| 0x1400     | LoaderUpdateFailed: The loader found an update but was unable to install it.                        | 1. Check your internet connection<br>2. Try a VPN service<br>3. Try manually reinstalling RusherHack using RusherHackInstaller                                                                                                                                                                                                                                                                                                                                                                                                                  |
| 0x1600     | InjectionFailed: Error injecting RusherHack into Minecraft                                          | 1. Make sure the game version matches your RusherHack version.<br>2. Try running RusherHack alone (without other mods). The issue may be due to incompatibility between mods and RusherHack.<br>3. Update your Java version.                                                                                                                                                                                                                                                                                                                    |
| 0x1650     | ClassLoaderError: Error initializing RusherHack files                                               | 1. Try a different launcher<br>2. Check for possible conflicts with other mods<br>3. Check your internet connection                                                                                                                                                                                                                                                                                                                                                                                                                             |
| 0x1800     | Error connecting to RusherHack servers                                                              | 1. Check your internet connection<br>2. Try using DNS: 1.0.0.1/1.1.1.1<br>3. Try using a VPN                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| No code    | The RusherHack and Minecraft versions don't match.                                                  | Check if you are using the correct version of Minecraft.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
